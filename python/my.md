# 자료형
* 순서 자료형             
    * 숫자(number)        
    * 문자열(strings)                
    * 리스트(Lists)               
    * 튜플(tuples)                      
                         
* 비순서 자료형             
    * 딕셔너리(dictionary)          
    * 집합(set)            
    * 부울(bool)           

## 순서 ) 숫자형(number)         
* 정수(integer) -> int()        
* 실수, 부동 소수점(floating point) -> float()          
* 데이터타입 출력 -> type(숫자)        
             
* 연산자     
|사칙연산|정수 나누기|나머지|제곱|                                 
|:------:|:---------:|:----:|:---:|                              
|덧셈 +|나머지 %|몫 //|제곱 ** |                       
|뺄셈 -||||             
|곱셈 * ||||                    
|나눗셈 /||||                                          
                          
* 관련 함수         
    * float()는 소수점이 있는 숫자를 소수점을 제외한 정수만 표시        
    * 절대값 -> abs()            
    * 제곱 -> pow(제곱하고 싶은 숫자, 제곱 횟수)
